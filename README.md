# Network Packet Analyzer

This project is a Python-based packet sniffer tool that captures and analyzes live network packets. It displays source and destination IP addresses, protocols, and packet information in real time.

## Features

* Captures live network packets
* Displays source and destination IP addresses
* Identifies network protocols
* Analyzes packet information in real time
* Helps understand network traffic flow

## Technologies Used

* Python
* Socket Programming

## How to Run

1. Install Python on your system.
2. Run the program:

```bash
python packet_sniffer.py
```
